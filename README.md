# laptop_price-regression
First attempt at machine learning with pandas and scikit-learn, aiming to improve personal programming skills.

## Overview
This program predicts laptops' prices based on features such as memory, cpu, etc.
- Firstly, I sperated the data (features and target) into a training group and a testing group
- Encoded both feature groups and reindexed one of them.
- Train the model
- Predict and compare
Right now, the results showed varied accuracy: some results are aligned whereas others may be mismatched.

## Libraries used
- Pandas
- Scikit-learn
